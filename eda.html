Project Overview

This project analyzes millions of Instacart orders to understand how customers shop for groceries online. Using the publicly available Instacart dataset, the analysis focuses on ordering patterns, products frequently purchased together, reorder behavior, and customer segmentation.

Example Visualization

The analysis produces several visual artifacts, including temporal order patterns, frequent itemset summaries, and network graphs of aisle-level associations.
